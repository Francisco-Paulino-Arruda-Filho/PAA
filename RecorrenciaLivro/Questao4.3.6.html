\( \text{Mostre que a solução para T(n) = \(2T(\frac{n}{2} + 17) + n \text{ é } O(n \log^n)\) } \)

\( \text{Pela definição de \(O\), temos que: \(\exists n_0, c_1 \) tal que \(T(n) \leq cg(n) \) } \)

\( T(n) \leq c(n \log^n) \)

\( T(n) = 2T(\frac{n}{2} + 17) + n \)

\( T(n) \leq 2c(\frac{n}{2} + 17) \log^{\frac{n}{2} + 17} + n \)

\( 2c(\frac{n}{2} + 17) \log^{\frac{n}{2} + 17} + n \leq c(n \log^n) \)

\( c(n + 34) \log^{\frac{n}{2} + 17} + n \leq c(n \log^n) \)

\( (\frac{n}{2} + 17)^{n+34} \leq n^n \)

\( (\frac{n}{2} + 17)^{n} * (\frac{n}{2} + 17 )^{34} \leq n^n \)

\( \text{Basta tomar \(c_0=1\) e \(n_0 \geq 68\)} \)
