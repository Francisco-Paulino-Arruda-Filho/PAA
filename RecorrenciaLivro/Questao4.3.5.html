\( T(n) = 2T(\lfloor \frac{n}{2} \rfloor) + n \)

\( \text{Enunciado Mostre que \(\Theta(n \log n)\) é a solução para a recorrência “exata” (4.3) para a ordenação por intercalação.} \)

\( \text{Pela definição de \(\Theta\), temos que: \(\exists n_0, c_1, c_2 \) tal que \(c_1g(n) \leq T(n) \leq c_2g(n) \) } \)

\( \text{Seja } T(n) = 2T(\lfloor \frac{n}{2} \rfloor) + n \)

\( T(n) \leq 2c_1\lfloor \frac{n}{2} \rfloor \log \lfloor \frac{n}{2} \rfloor + n \)

\( 2c_1\lfloor \frac{n}{2} \rfloor \log \lfloor \frac{n}{2} \rfloor + n \leq 2c_1\frac{n}{2} \log \frac{n}{2}+ n \)

\( c_1\frac{n}{2} \log \frac{n}{2}+ n \leq c_1n \log n \)

\( \text{Basta tomar \(c_1=1\) e \(n_0 \geq 4\)} \)

\( T(n) = 2c_2\lfloor \frac{n}{2} \rfloor \log \lfloor \frac{n}{2} \rfloor + n \)

\( T(n) \geq 2c_2\frac{n}{2} \log \frac{n}{2}+ n \)

\( 2c_2\frac{n}{2} \log \frac{n}{2}+ n \geq c_2n \log n \)

\( 2c_2\frac{n}{2} \log n - \frac{n}{2}\log^2 + n \geq c_2n \log n \)

\( c_2n \log n - \frac{n}{2}\log^2 + n \geq c_2n \log n \)

\( - \frac{n}{2}\log^2 + n \geq 0 \)

\( -\frac{n}{2} + n \geq n \)

\( n \geq 0 \)

\( \text{Basta tomar \(c_2=1\) e \(n_0 \geq 1\)} \)
