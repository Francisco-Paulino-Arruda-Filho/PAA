\( \text{Questão 03:} \)

\( \text{Mostre que a solução de \(T (n) = T (n - 1) + n \text{ é } O(n^2)\) } \)

\( \text{Pela definição de O, \(\exists c_1, n_0 \) tal que } T(n) \leq cg(n) \)

\( T(n) \leq c(n-1)^2 \)

\(\text{Resolução:}\)

\(T(n)=T(n-1) \)

\(T(n) \leq c(n-1-1)^2 \)

\(T(n) \leq c(n-2)^2 \)

\(T(n) \leq c(n^2 - 4n + 4) \)

\(c(n-1)^2 \leq c(n^2 - 4n + 4) \)

\(c(n^2-2n+1) \leq c(n^2 - 4n + 4) \)

\(cn^2-2cn+c \leq cn^2 - 4cn + 4c \)

\(c(n^2 - 4n + 4) \leq c(n-1)^2 \)

\(c(n^2 - 4n + 4) \leq c(n^2 - 2n + 1) \)

\(\text{Basta tomar \(c=1\) e \(n_0=3\) } \)

\(\text{Questão 05:} \)

\( \text{Enunciado:} \)

\(\text{Vimos que a solução de T (n) = 2T (⌊n/2⌋) + n é O(n lg n). Mostre que a solução}\)
\(\text{dessa recorrência é também Ω(n lg n). Conclua que a solução é Θ(n lg n).}\)

\(\text{Resolução:}\)

\( \text{Pela definição de Ω, \(\exists c_1, n_0 \) tal que } T(n) \geq cg(n) \)

\( T(n) \geq c n \log n \)

\( T (n) = 2T ( \lfloor n/2 \rfloor ) + n \)

\( T(n) \geq 2(c \lfloor \frac{n}{2} \rfloor \log \lfloor \frac{n}{2} \rfloor) + n \)

\( T(n) \geq 2(c \frac{n}{2} \log \frac{n}{2}) + n \)

\( T(n) \geq (cn \log \frac{n}{2}) + n \)

\( cn \log n \geq (cn \log \frac{n}{2}) + n \)

\( cn \log n \geq (cn \log^n - cn \log^2 ) + n \)

\( (cn \log^2 ) \geq n \)

\( \text{Basta tomar \(c=1\) e \(n_0=2\) } \)
