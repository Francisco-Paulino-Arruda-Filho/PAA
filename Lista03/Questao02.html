void buscar_binaria(int *v, int n, int x) {
                    int inicio = 1;
                    int fim = n;
                    int meio;
                    if(v[inicio] > x || v[fim] < x) {
                        return -1;
                    }
                    while (inicio <= fim) {
                        meio = (inicio + fim) / 2;
                        if (v[meio] == x) {
                            return meio;
                        } else if (v[meio] < x) {
                            inicio = meio + 1;
                        } else {
                            fim = meio - 1;
                        }
                    }
                    return -1;  
                }

\( \text{O que cada variável representa: n é o tamanho do vetor, v é o vetor e x o valor chave} \)

\( \text{Prova de corretude:} \)

\( \text{Caso base:} \)

\( \text{Se } n = 1 \text{, então } v[1] = x \text{ e a função retorna 1.} \)

\( \text{Caso indutivo:} \)

\(\text{Na linha do laço while, o índice é iniciado como 1, o vetor é incrementado enquanto inicio <= fim ou v[meio] != x,}\)
\(\text{ se ele encontrar o valor x, a função retorna meio, que é o índice do valor x no vetor.}\)
\(\text{Se o laço terminar sem encontrar o valor x, a função retorna -1,}\)
\(\text{Logo, independente do caso, a função retorna o índice do valor x no vetor ou -1 se não encontrar, pois graças ao while o índice é incrementado.}\)

\( \text{Terminação:} \)

\( \text{O laço while termina quando inicio > fim, ou seja, quando o vetor é percorrido completamente.} \)
